# Loading 加载

## 介绍
用于页面或模块的加载中状态。

## 用法
### 基础
```jsx
<OsLoading></OsLoading>
```
### 自定义尺寸
```jsx
<OsLoading size={60}></OsLoading>
```
### 自定义颜色
```jsx
<OsLoading color='blue'></OsLoading>
```



## API
### 属性
|参数|说明|类型|默认值|
|------|------|------|------|
|color|色值，可选|string|#DD1A21|
|size|尺寸，可选|number|34|
|className|自定义类名，可选|string|-|
|customStyle|自定义样式，可选|object|-|
